# E-commerce Customer Analysis

# Project Overview
This project analyzes e-commerce customer data to understand purchasing behavior, customer value, and engagement patterns.

# Objective
To identify high-value customers and derive actionable business insights using data analysis techniques.

# Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Skill Level Demonstrated
This project demonstrates intermediate data analytics skills, including schema inspection, debugging, metric-driven analysis, customer segmentation using percentiles, and business-focused interpretation of results.
